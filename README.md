# Data-analytics-project-1
Data analytics project of customer behavior dataset using tools like MYSQL, ms- excel, python, jupyer notebook in vs code and for visualisation Microsoft Power Bi



Overview

This project demonstrates a complete data analytics pipeline, starting from raw data to actionable insights. The workflow includes data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, and interactive dashboard development.

The goal of this project is to transform raw data into meaningful insights using modern analytics tools and present the results through a professional dashboard and presentation.

Project Workflow
1. Data Loading

Imported dataset using Python

Loaded data into a Pandas DataFrame for analysis

2. Exploratory Data Analysis (EDA)

Analyzed dataset structure

Generated summary statistics

Identified patterns, trends, and outliers

Visualized data using charts and graphs

3. Data Cleaning

Handled missing values

Removed duplicate records

Corrected inconsistent data types

Standardized data formats

4. SQL Analysis (MySQL Server)

Imported cleaned dataset into MySQL

Executed SQL queries to analyze key metrics

Used filtering, grouping, and aggregation functions

5. Power BI Dashboard

Connected Power BI with processed dataset

Built interactive dashboards

Added KPIs, charts, and slicers for better insights

6. Reporting & Presentation

Documented insights in a structured report

Created a presentation using Gamma to summarize key findings
